# Saral AI

AI-powered browser extension to reduce cognitive load and simplify web content.

## Features
- Cognitive Load Score (CLS)
- Focus Mode
- Adaptive UI
- Distraction filtering

## Setup

1. Clone repo
2. Open chrome://extensions
3. Enable Developer Mode
4. Load "extension" folder

## Usage
Click extension → Enable Focus Mode

## Tech Stack
- JavaScript
- Chrome Extension API
- DOM manipulation
- AI-ready architecture
